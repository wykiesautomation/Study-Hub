# Natasha's Study Hub Deployment Guide

## Steps:
1. Upload index.html to GitHub and enable GitHub Pages.
2. Create Google Sheet with tabs: DownloadQueue, Bundles.
3. Deploy Apps Script as Web App and set notify_url in index.html.
4. Create Google Docs invoice template and update TEMPLATE_ID.
5. Prepare Google Drive folder structure: StudyHub-Packs and Invoices.
6. Test PayFast integration using Sandbox.
